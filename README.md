# Plain Gulp starter template 

This repo is a gulp starter for starting projects. 

repo includes 

# gulp-sass
# browser-sync
# gulp-useref
# gulp-uglify
# gulp-if
# gulp-cssnano
# gulp-imagemin
# gulp-cache
# del
# run-sequence  

this starter requires node-sass to run correctly 

Remember to do run the `npm install` command after cloning this repo :) 

Be Sure to run npm install node-sass for gulp-sass dev dependency
